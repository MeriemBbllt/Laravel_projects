# Laravel E-Commerce & Wallet System

This project is a Laravel-based web application that combines an e-commerce store with a wallet and payment system simulation.

## Project Overview

The application includes multiple features such as an online store, checkout system, wallet management, and payment simulation tools.

## Main Features

### 🛒 Store Module
- Product catalog including different printer types
- Banners, covers, and other products
- Product categories and listing

### 🧾 Checkout System
- Shopping cart functionality
- Order summary and checkout process
- Order validation

### 💳 Wallet System
- User wallet management
- Add funds via card simulation
- Wallet balance tracking

### 🧮 Payment Calculator
- Basic payment simulation calculator
- Estimation of total cost before checkout

### 🔐 Security Features
- Secure authentication system
- User session management

## Technologies Used
- Laravel
- PHP
- MySQL
- Bootstrap / Frontend tools

## Purpose of the Project
This project was built for learning and practicing full-stack web development concepts including e-commerce workflows, wallet systems, and payment simulations.

## Note
Some features are simulated for educational purposes (payment system, wallet recharge).
